### Этап 1: Создание базового функционала

**Дата**: 2025-01-08

**Что было**: Нужно было создать простое решение для прослушивания lofi музыки без рекламы и задержек.

**Решение**: Создан простой веб-плеер с локальными MP3 файлами вместо стримов.

**Почему такое решение**:
- Локальные файлы загружаются мгновенно (без задержек)
- Нет зависимости от внешних стримов (которые могут быть нестабильны)
- Полный контроль над контентом
- Работает в фоне (HTML5 Audio API)

**Плюсы**:
- ✅ Быстрая загрузка треков
- ✅ Стабильная работа
- ✅ Без рекламы
- ✅ Работает в фоне
- ✅ Простой интерфейс (одна кнопка Play)

**Минусы**:
- ❌ Нужно хранить файлы (но 5-10 треков это нормально)
- ❌ Ограниченный набор треков (но можно добавлять)

**Подводные камни**: 
- GitHub не принимает файлы >100MB (пришлось исключить один большой трек)
- Нужен локальный сервер для разработки (из-за CORS при открытии через file://)
